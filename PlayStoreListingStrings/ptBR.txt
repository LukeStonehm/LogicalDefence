Defesa Lógica (Logical Defense)

Prepare-se para lutar contra os sofistas do mundo com este belo aplicativo.

Arme-se e proteja sua mente contra a linguagem utilizada pelos sofistas. 

Eles são muitos, nós somos poucos, mas com o aplicativo Defesa Lógica (Logical Defense) agora temos uma ferramenta extra para auxiliar nossa razão.
Utilize este aplicativo para rapidamente encontrar as falácias lógicas que lidamos ao debater contra o irracional.
Aconselhe aqueles que possuem falhas lógicas e ajude a construir uma sociedade mais racional.

Honestidade é a primeira virtude, então sejamos bem honestos com aqueles que tentarem utilizar a linguagem como uma forma de abuso mental. Portanto, nunca aceite uma falácia lógica, mas contra-ataque, e espalhe a lógica e a razão com aqueles que você ama.

Certo, chega de divagar. Este é um aplicativo que contém uma lista de falácias lógicas que assombram a mente de cada racionalista.

Algumas características:

- As categorias encontram-se à esquerda;
- Função compartilhar -> Mantenha pressionada uma falácia para compartilhar;
- Uma bela tela informativa;
- Função de seleção de idioma.

Este é um projeto totalmente livre (GNU v2), e o código está hospedado no Github e aceitamos colaborações :)
